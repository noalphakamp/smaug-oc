Ralph TUI is a terminal UI for orchestrating AI coding agents (Claude Code, OpenCode, etc.) to work through task lists autonomously. Features include extensible plugin architecture, built-in interactive PRD creator, auto conversion between tracker formats, customizable prompts, and understanding of task dependencies. Install via 'bun install -g ralph-tui' and initialize with 'ralph-tui init'.
