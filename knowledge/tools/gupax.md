# Gupax

> A GUI combining P2Pool and XMRig

## Overview

Gupax is a graphical user interface that combines P2Pool (a decentralized Monero mining pool) with XMRig (a popular Monero mining software). It provides an easy-to-use interface for mining Monero on a P2Pool node.

## Key Features

- **GUI Interface:** Visual interface for mining operations
- **P2Pool Integration:** Connect to decentralized P2Pool network
- **XMRig Integration:** Uses XMRig for RandomX hashing
- **Cross-Platform:** Available for multiple operating systems
- **Privacy-Focused:** Supports private mining operations

## Use Cases

- Individual Monero mining with GUI convenience
- P2Pool participation for decentralized mining
- Privacy-preserving cryptocurrency mining
