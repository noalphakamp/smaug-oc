DeepSeek's Engram introduces a new memory module for LLMs using modernized hashed N-gram embeddings for O(1) lookup-style conditional memory. The module adds an efficient scalable lookup capability that makes models effectively "deeper" for reasoning tasks by reducing the need for early-layer reconstruction of static patterns.
