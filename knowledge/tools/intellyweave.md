IntellyWeave is an open-source OSINT intelligence platform built on Elysia's decision tree architecture. Uses AI agents for document analysis, entity extraction, geospatial mapping, and network visualization. Demonstrated capability to map complex intelligence scenarios like Nazi escape routes across multiple continents.
