**Stars:** 5,065 | **Language:** TypeScript

**Topics:** deepresearch, deepsearch

## Description

Keep searching, reading webpages, reasoning until an answer is found (or the token budget is exceeded). Useful for deeply investigating a query.

## Implementation

Open-source implementation of Deep Research concept using Node.js, Gemini Flash, and Jina Reader for iterative web search and reasoning. The process loops through Search → Read → Reason until an answer is found or token budget exceeded.

## Architecture

```
Query → Search → Read → Reason → Search → ... → Answer
```
